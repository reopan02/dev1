# 用户指南

## 功能简介

### 竞品图片分析

上传竞品的电商图片，AI 会自动分析以下内容：

- **构图布局** - 产品在画面中的位置、大小比例、留白空间
- **光影效果** - 光源方向、阴影处理、高光分布
- **背景风格** - 背景颜色、纹理、场景元素
- **整体氛围** - 色调倾向、视觉风格、品牌调性

分析结果会生成一段详细的提示词，描述图片的视觉特征。

### 智能图片生成

基于分析得到的提示词，结合你上传的产品图片，AI 会生成同风格的电商图片。生成的图片会：

- 保持产品原有外观和细节
- 套用竞品图片的构图和风格
- 自动匹配光影和背景效果

### 参数配置

**宽高比选项：**

| 比例 | 适用场景 |
|------|----------|
| 1:1 | 淘宝主图、商品详情 |
| 16:9 | 横幅广告、PC 端展示 |
| 9:16 | 手机端详情页、短视频封面 |
| 4:3 | 传统商品展示 |
| 3:4 | 竖版商品图 |

**分辨率选项：**

| 分辨率 | 说明 |
|--------|------|
| 1K | 快速预览，适合测试效果 |
| 2K | 标准质量，适合大多数场景 |
| 4K | 高清输出，适合印刷或大图展示 |

## 使用方法

### 第一步：上传竞品图片

在左侧「竞品图片」面板上传参考图片。

**上传方式：**
- 点击上传区域选择文件
- 直接拖拽图片到上传区域
- 使用 Ctrl+V 粘贴剪贴板中的图片

**图片要求：**
- 格式：JPG 或 PNG
- 大小：不超过 5MB
- 建议：选择构图清晰、风格明确的竞品图

### 第二步：分析构图

点击「分析构图」按钮，等待 AI 分析完成。

分析过程中会显示加载状态，通常需要几秒到十几秒。完成后，提示词区域会显示分析结果。

### 第三步：编辑提示词

查看生成的提示词，可以根据需要进行修改：

- 调整背景描述（如改变颜色、添加元素）
- 修改光影效果（如增强对比度、改变光源方向）
- 补充特殊要求（如添加文字位置、品牌元素）

提示词支持中英文混合输入。

### 第四步：上传产品图片

在右侧「目标产品」面板上传你的产品图片。

**建议：**
- 使用白底或透明底的产品图
- 产品主体清晰完整
- 避免有复杂背景干扰

### 第五步：选择生成参数

根据用途选择合适的宽高比和分辨率。

**建议：**
- 首次生成选择 1K 分辨率快速预览
- 确认效果满意后再生成高分辨率版本

### 第六步：生成图片

点击「生成图片」按钮，等待 AI 生成完成。

生成时间取决于分辨率：
- 1K：较快
- 2K：中等
- 4K：较慢

### 第七步：下载保存

生成完成后，图片会显示在结果区域。

- 点击「下载图片」保存到本地
- 如不满意，可修改提示词后重新生成

## 使用技巧

### 获得更好的生成效果

1. **选择优质竞品图** - 构图专业、光影自然的图片分析效果更好
2. **使用干净的产品图** - 白底或透明底的产品图生成效果更佳
3. **适当调整提示词** - 根据产品特点微调提示词，如材质、颜色描述

### 批量生成建议

1. 先用 1K 分辨率测试多个竞品风格
2. 选定最佳风格后再生成高分辨率版本
3. 保存常用的提示词模板供后续使用

### 常见问题

**Q: 生成的图片产品变形了？**
A: 尝试使用更清晰的产品图，或在提示词中强调保持产品原有比例。

**Q: 背景效果不理想？**
A: 在提示词中详细描述期望的背景效果，或更换竞品参考图。

**Q: 生成速度很慢？**
A: 4K 分辨率需要更多处理时间，建议先用 1K 测试效果。
